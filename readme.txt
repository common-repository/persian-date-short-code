*  [Persian Support in Kanithemes.com](https://kanithemes.com)


== Installation ==

1. Upload `persian-date-shortcode` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. for display persian date in theme : Place `<?php do_shortcode('[wikiwordpress]'); ?>` in your templates
4. for display persian date in post,page or widget: use `[wikiwordpress]' with or without parameters
